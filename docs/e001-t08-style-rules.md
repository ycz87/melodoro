# E-001-T08 手绘风格统一（去拼装感）

## 本轮范围

仅做风格一致性收敛：笔触节奏、边缘软硬、颗粒密度；未做大构图与大比例调整。

## 风格规则清单（>=5）

1. **主描边统一粗细**：主要轮廓统一到 `2.2` 线宽，避免组件之间线重跳变。
2. **细节描边统一节奏**：次级细节线统一到 `1.4`，与主线形成稳定 1.57 倍关系。
3. **端点/转角统一**：图标 `strokeLinecap/strokeLinejoin` 统一为 `round`，去掉“硬折线拼接感”。
4. **高光软化一致**：地块顶面高光透明度从 `0.56` 收敛到 `0.52`，避免局部过亮跳出。
5. **颗粒密度统一**：顶面叠加低强度噪点（fractalNoise + alpha table），让块面拥有一致手绘颗粒。
6. **阴影色温统一**：配景椭圆阴影透明度微调为统一暖中性，不让单个组件显得过脏或过灰。
