# E-001-T07 四角配景比例与层级

## 本轮范围

仅调整四角配景的比例、位置、层级；未改地块纹理与阴影，未改变主构图中心。

> 尺寸列为 `w`（mobile/sm/md）。

## 层级与位置策略

1. **层级分组**：远景围栏/草丛在 `z-[9~10]`，中景建筑在 `z-[10~11]`，前景动物与近景围栏在 `z-[12~13]`。
2. **位置外扩但不过界**：上角主建筑向外扩 0.8%~1.0%，尺寸放大后仍保留边距，避免移动端裁切。
3. **前后景遮挡成立**：前景动物压在近景围栏之上，避免“层级不成立”的平贴感。
